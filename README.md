# bmpng

## Running

To use the command line interface:

```
python3 src/main.py ...
```

To launch the terminal user interface:

```
python3 src/tui.py
```

or:

```
python3 src/main.py tui
```

To launch the graphical user interface:

```
python3 src/gui.py
```

or:

```
python3 src/main.py gui
```
