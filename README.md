# bmpng

## Running

Without virtual environment:

```
$ cd src
$ python -m bmpng ...
```

With virtual environment:

```
$ python3 -m venv .venv # only needs to be done once
$ . .venv/bin/activate # only needs to be done once per session
$ pip install -e .
$ bmpng ...
```
